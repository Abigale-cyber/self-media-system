# 标题与内容包装

## 标题原则

Emma 的标题要有点击理由，但不能靠夸张承诺、容貌焦虑或标题党。

| 原则 | 说明 |
| --- | --- |
| 痛点具体 | 对应真实咨询问题，而不是泛泛说“变美” |
| 判断明确 | 给用户一个可理解的判断角度 |
| 风险克制 | 可以提醒风险，但不恐吓 |
| 专业可信 | 能和正文里的逻辑、案例或流程对应 |
| 平台适配 | 小红书偏场景收藏，视频号偏口播钩子，公众号偏深度判断 |

## 推荐标题类型

| 类型 | 用法 |
| --- | --- |
| 判断型 | “眼周显老，先判断是哪一类问题” |
| 避坑型 | “做项目前，先问清楚这几个恢复期问题” |
| 反常识型 | “有些眼袋问题，不是越早做越好” |
| 清单型 | “面诊前可以准备的 5 个问题” |
| 温和共鸣型 | “你不是突然变老，可能是疲惫感被放大了” |

## 禁用标题倾向

- 不用“逆袭、换脸、年轻十岁、一次解决、永久有效”。
- 不用“再不做就晚了、不做会毁脸”等焦虑表达。
- 不用“最安全、唯一方案、人人适合”等绝对化表达。

## 修改顺序

1. 先确认标题是否和核心判断一致。
2. 再确认标题是否具体到目标用户。
3. 最后适配平台长度、封面文字和摘要。
