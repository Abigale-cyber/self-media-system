# 账号与渠道表达安全边界

本文件是内容生产和平台适配时的内部风险提示，不是法律意见，也不替代发布平台的最新规则核验。

## 通用边界

- 医美内容不写确定疗效、绝对安全、必做项目或容貌焦虑刺激。
- 没有面诊和完整信息，不给个体诊断、项目选择结论、治疗参数或恢复保证。
- 评论区以观点互动、一般性知识和就医提醒为主，不做在线问诊。
- 直播或视频互动用于认知分享，不做实时诊断、个案方案或价格导流。

## 账号与平台侧重点

| 场景 | 推荐表达 | 避免表达 |
| --- | --- | --- |
| 认证医生号视频 | 观察到的问题、判断条件、风险提醒、何时需要面诊 | 直接项目推荐、固定效果、强转化 CTA |
| 非认证人格号视频 | 创业/生活观察、女性状态、长期判断 | 以医生身份给方案、AI 赚钱承诺、认知大师语气 |
| 小红书 | 具体场景、误区拆解、可收藏判断清单 | 夸张前后对比、容貌焦虑标题、隐性导流 |
| 视频号 | 清楚的口播观点、温和互动、长期信任 | 在线问诊式评论回复、价格或预约引导 |
| 公众号 | 完整逻辑、证据来源、风险边界和阅读体验 | 无依据的医疗结论、只讲效果不讲限制 |

## 发布前动作

1. 明确账号模式和发布平台。
2. 对医美主题运行 `medical-aesthetic-compliance-checker`。
3. 对可能受平台规则变化影响的发布表达，发布前另行核验当前平台要求。
