# Emma 自媒体选题评分标准

`topic-generator` 必须使用这套规则进行选题评分和排序。

## 评分维度

每个候选选题按 7 个维度评分：

| 维度 | 分值 | 判断标准 |
| --- | ---: | --- |
| 痛点强度 | 1-5 | 是否对应真实高频问题、犹豫点、误区、决策焦虑或门诊常见问题 |
| 业务价值 | 1-5 | 是否能自然服务本账号的目标，例如专业信任、同行影响、人格陪伴或用户认知 |
| 传播潜力 | 1-5 | 是否有反差、收藏价值、讨论度、平台标题空间或短视频钩子 |
| 专业可信度 | 1-5 | 是否能用提供的专业资料、判断逻辑、场景证据或风险提醒支撑 |
| 账号匹配度 | 1-5 | 是否适合本期认证医生号或非认证人格号的定位、受众和表达边界 |
| 资料充分度 | 1-5 | 是否已有可引用的知识库片段、用户问题和核心判断支撑成稿 |
| 合规风险 | 1-5 | 1 为低风险，5 为高风险；疗效承诺、焦虑营销、前后对比、诊疗建议会升高风险 |

综合分：

```text
痛点强度*4 + 业务价值*4 + 传播潜力*3 + 专业可信度*3 + 账号匹配度*2 + 资料充分度*2 + (6-合规风险)*2
```

## 档位

| 综合分 | 档位 | 处理方式 |
| ---: | --- | --- |
| >= 80 | 优先做 | 进入 `content-outline-builder` |
| 65-79 | 补素材后做 | 先补知识库片段、客户问题或专业依据 |
| 50-64 | 观察或合并 | 合并同类选题，等待更强场景 |
| < 50 | 暂不做 | 不进入本轮生产 |

## 医美降级规则

- 合规风险为 5 且无法改角度：淘汰。
- 痛点强度低于 3：不进入 Top 3，除非存在明确热点窗口。
- 专业可信度低于 3：改成轻量问答或风险提醒，不做强观点。
- 账号匹配度低于 3：换账号模式或重新选择角度后再评。
- 资料充分度低于 3：不得进入写稿流程，先回到 `topic-generator` 补资料。
- 涉及效果、适应症、恢复期、对比案例时，优先改成“判断方法、风险提醒、面诊问题清单”。

## 审稿评分

`content-reviewer` 使用 10 分制：

| 维度 | 分值 |
| --- | ---: |
| 标题吸引力 | 0-2 |
| 结构稳定性 | 0-2 |
| 论据强度 | 0-2 |
| 用户价值 | 0-2 |
| 表达完成度 | 0-2 |

| 总分 | 结论 |
| ---: | --- |
| >= 8 | 通过 |
| 5-7 | 需修改 |
| < 5 | 需重写 |
