# Emma 内容开头与结尾规则

## 开头原则

开头要让目标用户知道“这和我有关”，但不能靠羞辱外貌、夸大风险或制造焦虑来抓注意力。

| 类型 | 适合场景 | 示例方向 |
| --- | --- | --- |
| 场景提问 | 用户常见困扰 | “为什么眼周看着累，不一定先看项目？” |
| 反常识 | 纠偏类内容 | “有些眼周问题，急着做项目反而会绕远。” |
| 判断清单 | 小白决策 | “面诊前，先把这 3 个问题问清楚。” |
| 风险提醒 | 高风险误区 | “这类恢复期说法，听起来轻松，但要分情况。” |
| 案例切入 | 用户提供真实素材 | “上周有个咨询者问了一个很典型的问题……” |

## 结尾原则

结尾要给出下一步判断方法，而不是强成交。

| 类型 | 用法 |
| --- | --- |
| 总结判断 | 用一句话收束核心观点 |
| 面诊问题清单 | 告诉用户面诊时该问什么 |
| 风险提醒 | 补充个体差异、恢复期和禁忌 |
| 温和互动 | 邀请用户描述自己的情况，不做线上诊断 |

## SCQA

| 缩写 | 含义 | 在医美内容里的作用 |
| --- | --- | --- |
| S | 情境 | 用户熟悉的变美、恢复、咨询或选择场景 |
| C | 冲突 | 用户误区、期待和真实条件之间的矛盾 |
| Q | 问题 | 读者真正需要判断的问题 |
| A | 答案 | Emma 给出的专业判断方法和边界 |
