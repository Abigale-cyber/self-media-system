# 发布前检查清单

## 内容质量

1. 标题承诺的内容正文已经兑现。
2. 核心判断前后一致，没有突然换观点。
3. 至少有 3 条可用素材，且来源已标注。
4. 目标用户具体，不是“所有想变美的人”。
5. 术语有解释，小白能看懂。
6. 文章或口播没有明显 AI 套话。

## 医美风险

1. 没有“保证、永久、立刻见效、无风险、人人适合”等表达。
2. 没有用容貌羞辱或焦虑刺激推动成交。
3. 没有把个体情况说成通用结论。
4. 涉及项目、恢复期、适应人群时，保留面诊和个体差异提醒。
5. 案例和前后对比没有暗示固定效果。
6. 已调用或建议调用合规检查（`medical-aesthetic-compliance-checker`）。

## 平台发布

| 平台 | 发布前确认 |
| --- | --- |
| 小红书 | 标题、封面文字、标签和评论引导不过度功效化 |
| 视频号 | 前 3 秒字幕不制造恐慌，CTA 不强转化 |
| 公众号 | 标题、摘要、封面图、排版和图片位置已在 `wechat-studio` 预览 |

## 公众号工作台

- 发布前至少预览一遍手机样式。
- 未配置草稿箱推送时，只保留本地预览，不自动发布。
- 标题和封面确认后再推送；公众号发布后标题修改空间有限。
