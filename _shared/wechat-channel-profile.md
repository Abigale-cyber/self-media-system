# 公众号渠道规范

## 排版默认值

| 项目 | 默认 |
| --- | --- |
| 字号 | 15px |
| 字体颜色 | #333333 |
| 加粗颜色 | #2f6f9f |
| 行高 | 2 |
| 段落 | 手机阅读友好，每段不超过 5 行 |

## 标题和摘要

- 标题要兑现正文承诺，不做夸张功效暗示。
- 摘要不写全文总结，优先写一句能说明价值的判断。
- 封面图和标题方向要一致，避免“医美功效前后对比”式强刺激。

## 配图

优先使用：

1. 用户确认可公开的真实场景图或生活图。
2. `content-image-gen` 生成的安全风格封面和配图。
3. 无版权风险的公开素材。

配图避免：

- 夸张前后对比。
- 暗示固定效果。
- 未授权真人隐私。
- 明显医疗诊断或治疗场景误导。

## 发布提醒

- 公众号导入和预览交给公众号工作台（`wechat-studio`）。
- 推送草稿箱前必须人工确认标题、摘要、封面和正文图片。
- 未配置公众号发布能力时，只输出 Markdown、HTML 预览和发布包。
