# 医美选题数据源手册

## 数据源优先级

| 优先级 | 来源 | 用途 |
| --- | --- | --- |
| 1 | `topic-material-pack.md` 中 Emma 已提供的资料片段与客户问题 | 找真实痛点和可支撑判断的材料 |
| 2 | `knowledge-base-index.md` 指引 Emma 补充的业务模块和常见问题资料 | 对齐眼周抗衰、痘肌/问题肌、生活美学和账号模式 |
| 3 | 小红书、视频号、公众号竞品内容 | 找标题、评论痛点和平台表达方式 |
| 4 | 权威科普、行业信息、平台规则 | 校准专业说法和风险边界 |
| 5 | 联网搜索和热点榜单 | 找近期讨论窗口，不能替代专业判断 |

## 搜索模式

### A. 痛点模式

适合从客户问题和评论区找选题。

关键词组合：

- `眼周显老 + 原因 / 面诊 / 恢复期 / 误区`
- `眼袋 + 适合人群 / 风险 / 术后 / 问题`
- `痘肌 + 反复 / 屏障 / 医美 / 护理误区`

### B. 竞品模式

适合看平台上已经被验证的表达。

检查：

- 标题在承诺什么
- 评论区反复问什么
- 用户收藏的原因是什么
- 哪些表达有夸大或合规风险

### C. 热点借势模式

只在热点能和 Emma 的专业边界自然连接时使用。

判断：

- 这个热点是否和医美决策、审美趋势、风险认知有关
- Emma 是否能提供专业判断，而不是复述热闹
- 是否会引发不必要的焦虑或争议

## 选题搜集流程

1. 先从资料盘点结果提取候选题；需要扩展时再从外部来源各提取 2-3 个信号。
2. 合并相似题目，去掉明显重复。
3. 删除太宽泛、太功效化、太焦虑、无观点的题。
4. 按 `../_shared/scoring-rubric.md` 初评分。
5. 对高分题做角度重写，输出 `topic-pool.md`。

## 信息标注

每个原始信号必须标注：

- 平台或资料来源
- 证据摘要
- 可信度：高 / 中 / 低 / 待验证
- 是否需要合规复核

无法验证的信息可以作为“待验证线索”，不能作为确定事实写入正文。
