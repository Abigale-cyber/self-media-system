# Emma 医美选题原则

## 好选题公式

好选题 = 真实痛点 × 专业判断 × 可传播表达 × 合规可控

| 维度 | 说明 |
| --- | --- |
| 真实痛点 | 来自咨询、评论、客户原话或高频误区 |
| 专业判断 | Emma 能提供分层解释、判断方法或风险提醒 |
| 可传播表达 | 标题和开头有场景、有反差、有收藏价值 |
| 合规可控 | 不依赖疗效承诺、前后对比或焦虑营销 |

## 三类优先选题

### A. 判断方法型

帮助小白知道“我属于哪种情况、该怎么判断”。

示例方向：

- 眼周显老，不一定先看项目，先判断原因。
- 痘肌反复，先区分屏障问题还是炎症问题。

### B. 风险避坑型

帮助用户在面诊、恢复期、项目选择前少走弯路。

示例方向：

- 面诊前需要问清楚的恢复期问题。
- 哪些“无恢复期”说法需要谨慎听。

### C. 认知纠偏型

把平台上常见但不准确的说法拆开讲清楚。

示例方向：

- 不是所有眼袋问题都适合用同一种处理思路。
- 不要把“贵项目”直接等同于“适合你”。

## 反模式

| 反模式 | 问题 |
| --- | --- |
| 太宽泛 | “医美入门指南”无法支撑强观点 |
| 太功效化 | 容易变成效果承诺 |
| 太焦虑 | 会损害信任，也有合规风险 |
| 太专业黑话 | 小白看不懂，难传播 |
| 无 Emma 判断 | 只是搬运平台热词 |
| 案例不可公开 | 后续写作和发布无法复用 |

## 角度重写

重写公式：

- 原问题 + “真正要先判断的是” + 分层标准
- 原误区 + “但更稳妥的说法是” + 风险边界
- 原热点 + “对面诊决策的启发是” + 可执行问题清单

重写后必须能回答：用户看完这条内容，下一次咨询或面诊会更会问什么。
