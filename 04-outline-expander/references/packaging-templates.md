# 标题、开头和结尾优化模板

## 标题模板

| 类型 | 适用场景 | 示例结构 |
| --- | --- | --- |
| 判断型 | 需要先纠正常见误区 | `不是 X，而是先判断 Y` |
| 避坑型 | 适合医美风险提醒 | `做 X 前，先问清楚 Y` |
| 反常识型 | 有明确反差判断 | `有些 X，不是越早做越好` |
| 清单型 | 适合小红书拆条或公众号小节 | `判断 X，先看这 N 点` |
| 温和共鸣型 | 适合焦虑人群 | `你担心的 X，可能先要看 Y` |

## 开头模板

| 类型 | 写法 |
| --- | --- |
| 场景切入 | 从门诊、咨询或日常观察中的高频问题写起。 |
| 误区切入 | 先点出用户最容易混淆的一件事，再给判断方向。 |
| 问题切入 | 用一个具体问题承接标题，不做夸张恐吓。 |
| 医生观察 | 用“我更关注的是……”呈现专业判断顺序。 |
| 反常识切入 | 温和指出“不一定是你想的那样”。 |

## 结尾模板

| 类型 | 写法 |
| --- | --- |
| 判断框架 | 总结读者可以按什么顺序判断。 |
| 风险提醒 | 回到面诊、个体差异和不要急着做项目。 |
| 留白型 | 用一句克制判断收束，不强导流。 |
| 复盘型 | 回扣开头场景，说明真正该先看什么。 |
