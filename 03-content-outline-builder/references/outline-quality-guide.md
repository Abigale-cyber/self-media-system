# Content Outline 质量标准

## 最关键的字段：文章大纲

大纲决定后续能不能顺利写成公众号文章、口播和工作台预览稿。`推荐框架` 只是文章类型，不能替代章节级大纲。

合格大纲必须同时回答：

- 这篇写给谁看
- 每一节解决读者什么问题
- 每一节用什么素材支撑
- 章节之间为什么按这个顺序推进
- 哪些表达需要合规复核

## 切入角度

好角度：

- 一句话说清楚。
- 有明确立场，不是“从多个角度分析”。
- 和 Emma 的专业判断有关。
- 有用户原话、案例、流程或风险提醒支撑。

示例方向：

- “眼周显老，不要先问做什么项目，先判断是哪一类问题。”
- “所谓无恢复期，真正要问的是恢复边界和个体差异。”

不合格角度：

- “眼周抗衰全面指南”
- “医美小白必须知道的一切”
- “某项目效果怎么样”

## 反模式

| 反模式 | 问题 | 修复 |
| --- | --- | --- |
| 只有标题没有章节 | 后续写手不知道怎么展开 | 增加每节要回答的问题、关键内容和素材 |
| 读者太泛 | 内容容易变成科普大全 | 缩窄到具体人群和场景 |
| 没有核心判断 | 文章会变成信息搬运 | 用一句话写出明确立场 |
| 没有素材来源 | 容易编事实 | 至少 3 条素材，标注来源和可信度 |
| 没有风险提醒 | 医美内容容易踩线 | 加入合规边界和替代表达方向 |
| 只有项目介绍 | 容易功效化 | 改成判断方法、面诊问题或风险清单 |

## 自检

1. 核心判断够硬？
2. 目标用户够具体？
3. 素材至少 3 条且来源清楚？
4. SCQA 完整？
5. 文章大纲独立存在，且每节可写？
6. 覆盖题目承诺？
7. 有医美风险提醒？

任何一项不通过，回到大纲阶段修改。
