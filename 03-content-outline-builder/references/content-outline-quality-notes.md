# Content Outline 质量补充

## 检验方法

两个不同写手拿到这份 content-outline，应该能写出方向一致、章节顺序接近、核心判断不跑偏的稿件。

## 必须具备

| 项目 | 要求 |
| --- | --- |
| 核心判断 | 一句话，有立场，可被全文反复支撑 |
| 目标用户 | 具体到场景和认知水平 |
| 文章大纲 | 5-10 个一级章节，每节有问题、内容和素材 |
| 素材清单 | 至少 3 条，标注来源和可信度 |
| 风险提醒 | 明确哪些表达不能写、哪些需要合规检查 |

## 医美内容特别要求

- 不把“项目介绍”当成大纲。
- 不把“适合所有人”当成结论。
- 不用前后对比或效果承诺支撑观点。
- 涉及恢复期、适应人群、术式选择时，改成面诊问题、判断条件和风险提醒。

## 输出前自检

1. 角度够具体？
2. 章节能直接写？
3. 素材能公开使用？
4. 风险边界清楚？
5. 下一步 Skill 能直接接上？
