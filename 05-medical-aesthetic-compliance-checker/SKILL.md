# medical-aesthetic-compliance-checker

用于对医美自媒体内容做发布前风险审查。它不是法律意见，不替代律师、平台审核或医疗合规人员判断。

执行前读取 `../_shared/knowledge-base-index.md`、`../_shared/account-mode-guide.md`、`../_shared/channel-safety-guide.md`、`../_shared/style-guide.md` 和 `references/risk-lexicon.md`。如果 Emma 尚未提供合规原文或平台规则，按更保守的内容风险清单审查并提示发布前再核验。

## 使用范围

使用这个 Skill 处理：

- 用户说“做医美合规检查”
- 公众号、视频号、小红书、口播稿发布前复核
- 内容涉及项目效果、适应人群、案例对比、术后恢复、医生/机构资质
- 需要把高风险表达改成更安全的表达

如果稿件尚未成型，先建议完成 `content-polisher` 或 `adversarial-content-review` 后再审。

## 风险清单

逐项检查：

| 风险类型 | 高风险表现 |
| --- | --- |
| 疗效承诺 | 保证变美、一定改善、永久有效、立刻见效、无恢复期 |
| 绝对化表达 | 最安全、唯一、首选、100%、人人适合、没有副作用 |
| 焦虑营销 | 不做就老、脸垮、毁脸、输在起跑线、必须马上处理 |
| 前后对比 | 暗示固定效果、使用夸张对比、未说明个体差异 |
| 资质暗示 | 暗示非医生可诊断/治疗，或夸大机构/专家背书 |
| 诊疗建议 | 未面诊就给出项目方案、剂量、治疗结论 |
| 风险提示缺失 | 只讲好处，不讲禁忌、恢复、个体差异和面诊必要性 |
| 平台表达风险 | 过度功效词、强转化 CTA、价格诱导、案例真实性不清 |

## 审查流程

1. 先判断内容类型：科普、种草、案例、口播、平台发布包。
2. 判断账号模式：认证医生号 / 非认证人格号 / 待确认；不清楚时按更保守口径处理。
3. 判断目标平台：公众号 / 小红书 / 视频号 / 多平台 / 待确认。
4. 逐句扫描高风险表达，记录原句。
5. 判断风险等级：高 / 中 / 低。
6. 给出原因和替代表达。
7. 同时检查标题、封面文字、字幕、正文、评论引导和 CTA，而非只检查正文。
8. 如果改写不会改变原意，输出合规版全文；如果资料不足，只输出局部替换建议。

### 合规改写原则

输出适用于这篇稿件的 3-5 条改写原则，例如：

- 把“保证效果”改成“可能改善，需结合基础条件判断”
- 把“必须做”改成“可以作为面诊时讨论的方向”
- 把“无风险”改成“需要了解恢复期、禁忌和个体差异”

### 合规版全文

当原稿完整且可改写时输出。改写要尽量保留 Emma 的专业表达和内容价值，不要把文章改成免责声明堆砌。

同时按 `../_shared/intermediate-format.md` 中 `compliance-report.md` 的结构输出可交接版本，供 `platform-adapter` 读取。

## 判断标准

- PASS：没有明显高风险表达，只需轻微措辞优化。
- NEEDS EDIT：存在中低风险表达，改写后可继续发布。
- HIGH RISK：存在明确疗效承诺、严重焦虑营销、诊疗结论或资质风险，建议重写后再审。
- 非认证人格号出现项目推荐、诊疗口吻或效果判断时，至少判为 NEEDS EDIT；无法移除时判为 HIGH RISK。
