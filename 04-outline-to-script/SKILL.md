# outline-to-script

用于把已经确认的内容大纲转成可录制的短视频口播稿和轻量分镜。它不写公众号长文，也不负责生成图片或发布。

执行前读取 `../_shared/knowledge-base-index.md`、`../_shared/account-mode-guide.md`、`../_shared/channel-safety-guide.md`、`../_shared/style-guide.md` 和 `../_shared/intermediate-format.md`，保证口播稿延续 Emma 的账号身份、专业边界和内容大纲。

## 使用范围

使用这个 Skill 处理：

- 用户说“根据这个大纲生成口播稿”
- 用户输入 `/口播`
- 已有 `content-outline`，不想先写长文，想直接做视频
- 需要 60/90/180 秒版本、封面标题或分镜建议
- 需要把专业内容改成小白能听懂的医美口播

如果没有大纲或核心素材，先向用户要：主题、目标受众、核心观点、必须保留的信息和禁讲边界。

## 生成原则

- 默认优先输出 90 秒版本；用户指定时长时按用户要求。
- 口播要一句话一个重点，短句为主，减少书面语。
- 开头 3-5 秒必须给观看理由，但医美内容不能靠制造容貌焦虑吸引注意。
- 保留专业判断链：问题是什么、为什么会发生、怎么判断、什么时候需要面诊。
- 允许有情绪和共鸣，但不能夸大痛点、承诺效果或暗示“不做就会变差”。
- 如果引用案例，只能使用用户提供的案例；没有案例就写成“常见情况/沟通场景”。

## 账号模式结构

| 账号模式 | 主结构 | 特别限制 |
| --- | --- | --- |
| 认证医生号 | 问题 -> 原因 -> 判断条件 -> 风险与面诊提醒 | 可以表达医生判断，不在线诊断或推荐固定方案。 |
| 非认证人格号 | 观察 -> 共鸣 -> 认知 -> 留白 | 以创业、生活或女性状态观察为主，不使用诊疗语气。 |

账号模式为待确认时，先向 Emma 确认；无法确认而必须出草稿时，按非认证人格号的保守边界输出并显式标注。

## 可选结构

根据大纲选择一种，不要机械套用：

| 结构 | 适合场景 |
| --- | --- |
| 痛点误区 -> 原因 -> 判断方法 -> 提醒 | 用户有明确困扰或常见误解 |
| 结果前置 -> 条件拆解 -> 风险提醒 -> 行动建议 | 用户想知道某项目是否适合 |
| 反常识观点 -> 解释 -> 案例/场景 -> 总结 | 需要做专业认知纠偏 |
| 问题清单 -> 分层回答 -> 决策建议 | 适合小白科普和收藏型内容 |

## 输出格式

同时按 `../_shared/intermediate-format.md` 中 `voice-script.md` 的结构，输出可交给审稿、合规和平台适配的口播脚本文档。

- 目标平台：
- 建议时长：
- 核心观点：
- 钩子类型：引发好奇 / 直击痛点 / 警告避坑 / 价值展示 / 反向纠偏
- 合规提醒：

### 开头钩子备选

输出 3 条，每条不超过 35 字，并说明适合原因。

### 口播逐字稿

按用户指定时长输出。若用户没有指定，输出 90 秒版本。正文只写可直接录制的逐字稿，不加小标题和项目符号。

### 分镜建议

| 镜头 | 时长 | 画面内容 | 字幕重点 | 音乐/音效 | 对应口播 |
| ---: | --- | --- | --- | --- | --- |

分镜只做轻量建议，避免复杂剪辑自动化、自动视频生成或外部工具流程。

### 封面标题

输出 5 个备选，分为：

- 避坑型
- 判断型
- 反常识型
- 收藏型
- 温和共鸣型

### 发布前风险表达

列出稿件中可能需要交给 `medical-aesthetic-compliance-checker` 复核的句子或表达方向。

## 最小测试

- 输入“根据这个大纲生成短视频口播稿”，应输出 90 秒逐字稿、分镜建议、封面标题和风险表达。
- 输入“/口播”，应优先触发本 Skill；如果用户给的是完整文章而不是大纲，转给 `voice-script-writer`。
- 涉及项目效果、恢复期或适应人群时，应保留面诊和个体差异提醒。

## 禁用倾向

- 不使用“保证、永久、立刻见效、无风险、最有效、必做、逆袭、换脸”等绝对或夸大表达。
- 不用羞辱外貌、制造恐慌或暗示不做项目就会失败的表达。
- 不把医生面诊、医学诊断和治疗建议简化成通用结论。
