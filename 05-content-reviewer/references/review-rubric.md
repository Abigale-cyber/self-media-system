# 内容审稿评分标准

## 三角色

| 角色 | 关注点 |
| --- | --- |
| 笔杆子 | 核心观点是否清楚，章节是否各自交付价值，论据是否支撑判断，是否有逻辑跳跃或空泛升华。 |
| 参谋 | 目标读者是否愿意看完，哪里看不懂或想跳过，哪里有共鸣、收藏和转发价值。 |
| 裁判 | 综合内容质量与风险边界，给出五维度评分、结论和优先修改清单。 |

## 五维度评分

| 维度 | 满分 | 判断重点 |
| --- | ---: | --- |
| 标题吸引力 | 2 | 标题是否具体、有判断、不靠焦虑或夸张承诺。 |
| 结构稳定性 | 2 | 开头、章节推进和结尾是否顺，是否兑现题目承诺。 |
| 论据强度 | 2 | 是否有资料、场景、专业判断或风险提醒支撑。 |
| 用户价值 | 2 | 读者看完是否知道如何判断、避开什么误区或下一步怎么做。 |
| 表达完成度 | 2 | 语气是否像 Emma，是否克制、真实、少 AI 痕迹。 |

| 总分 | 结论 |
| ---: | --- |
| >= 8 | 通过 |
| 5-7 | 需修改 |
| < 5 | 需重写 |

## 修改优先级

| 优先级 | 定义 |
| --- | --- |
| P0 | 不改不建议发布：核心观点不成立、高风险表达、账号越界或明显误导。 |
| P1 | 发布前建议修改：论据不足、结构跳跃、开头弱、重点不清。 |
| P2 | 可选优化：标题更聚焦、句子更短、CTA 更自然、语气更像 Emma。 |
