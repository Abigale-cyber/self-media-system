# 医美内容风险词表

## 高风险类型

| 风险类型 | 高风险表现 | 更稳妥方向 |
| --- | --- | --- |
| 疗效承诺 | 保证、一定改善、永久有效、立刻见效、一次解决、无恢复期 | 可能改善的方向、需要结合基础条件判断、结果存在个体差异 |
| 绝对化表达 | 最安全、唯一、首选、100%、人人适合、没有副作用、无风险 | 需要了解风险、禁忌、恢复安排，并由医生判断 |
| 焦虑营销 | 不做就老、脸垮、毁脸、输在起跑线、必须马上处理 | 很多人会在意这种变化，可以先了解原因和判断路径 |
| 前后对比 | 暗示固定效果、夸张对比、未说明个体差异 | 案例只作为个体情况，不代表普遍结果 |
| 资质暗示 | 暗示非医生可诊断或治疗，夸大机构、医生或设备背书 | 回到专业评估、合规资质和个体差异边界 |
| 诊疗建议 | 未面诊给项目方案、剂量、治疗结论 | 类似表现可能有不同原因，具体情况需要专业评估 |
| 风险提示缺失 | 只讲好处，不讲禁忌、恢复、个体差异和面诊必要性 | 增加判断条件、风险提醒和面诊问题 |
| 平台表达风险 | 价格诱导、强 CTA、案例真实性不清、过度功效词 | 以认知分享、判断清单和温和互动替代 |

## 结论分级

| 等级 | 判断 |
| --- | --- |
| PASS | 没有明显高风险表达，只需轻微措辞优化。 |
| NEEDS EDIT | 存在中低风险表达，改写后可继续。 |
| HIGH RISK | 存在明确疗效承诺、严重焦虑营销、诊疗结论或资质风险，建议重写后再审。 |
